# منصة خبرات (Khebrat Platform)

منصة متكاملة لإصدار وتوثيق الشهادات المهنية بشكل احترافي، تتيح للشركات والجهات إصدار الشهادات الفردية والجماعية (عبر ملفات Excel)، مع توفير روابط تحقق وباركود لضمان موثوقية الشهادات.

## المميزات الرئيسية 🌟
- **الإصدار الفردي والجماعي:** إصدار شهادات للموظفين بشكل فردي، أو رفع ملف Excel لإصدار مئات الشهادات بضغطة زر.
- **الإرسال التلقائي:** إرسال رابط الشهادة مباشرة إلى البريد الإلكتروني للمستفيد (Email Integration).
- **لوحة تحكم احترافية:** إدارة كاملة للشهادات، ترتيبها، إبطالها، أو إيقافها، مع إحصائيات متكاملة للشركات والمسؤولين.
- **التحقق الفوري:** صفحة تحقق خاصة بكل شهادة مع QR Code متوافق مع كافة الأجهزة.
- **تصميم ديناميكي ومخصص:** إمكانية تخصيص الألوان، رفع شعار الجهة، وختم الاعتماد، وتصدير الشهادة بصيغة PDF أو كصورة احترافية.

## التقنيات المستخدمة 🛠️
- Next.js (React)
- Tailwind CSS & Framer Motion (للتصميم والحركات)
- Firebase (قاعدة البيانات والمصادقة)
- Resend (لإرسال البريد الإلكتروني)
